### 💳 [Wallet API](https://github.com/dennismugane/Wallet-API) — Production-Grade Fintech Backend

> A high-concurrency digital wallet system built for financial-grade reliability

```
Java 21 · Spring Boot 3.4 · PostgreSQL · Redis · Docker · Kubernetes · GitHub Actions
```

| Feature | Implementation |
|---|---|
| 🔒 Concurrency Safety | PostgreSQL pessimistic locking + Redis distributed locks |
| 🔁 Idempotency | Custom request headers prevent duplicate transactions |
| 📒 Audit Trail | Append-only transaction ledger (double-entry logic) |
| 🛡️ Security Pipeline | Trivy CVE scanning + Semgrep SAST on every commit |
| 📊 Observability | Prometheus metrics + Grafana dashboards |
| ☸️ Production-Ready | Kubernetes manifests with rolling deployments + auto-rollback |
